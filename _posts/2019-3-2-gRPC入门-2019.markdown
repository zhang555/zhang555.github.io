## gRPC

gRPC是远程过程调用的一种实现，使用HTTP/2协议

使用Protocol Buffers作为接口描述语言

提供了认证，双向流，流控制，阻塞非阻塞绑定，取消，超时等特性

它生成跨平台的client和server

常用于微服务，移动设备连接，浏览器和后端服务


## 基本使用

先定义pb文件，然后使用protoc命令生成go代码，最后执行go程序即可。

protoc命令要指定使用grpc插件

```
protoc -I helloworld/ helloworld/helloworld.proto --go_out=plugins=grpc:helloworld

go run greeter_server/main.go
```
